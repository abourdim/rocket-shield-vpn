DNS stands for Domain Name System. It is like a phone book for the internet. Instead of remembering numbers like 142.250.80.46, you just type google.com and DNS finds the right number for you!

The 4 Types of DNS Servers

DNS Resolver — Your helper that does all the asking (usually your ISP).
Root Server — Knows who manages .com, .org, .fr, etc. (only 13 in the world!).
TLD Server — Manages one extension like .com or .org.
Authoritative Server — The final answer! Knows the exact IP address.

DNS Record Types

A — IPv4 address (e.g., 142.250.80.46)
AAAA — IPv6 address (e.g., 2607:f8b0:4004::200e)
MX — Mail server (where emails go)
CNAME — Alias pointing to another domain
TXT — Text info (used for verification)

Why DNS matters for privacy

Without protection, your ISP sees every single domain you visit!

Every time you type a website address, your device asks a DNS server to translate it. By default, that server belongs to your Internet Service Provider (ISP). This means they can log every website you visit.

How VPN DNS Protection Works

A VPN routes your DNS queries through an encrypted tunnel to a private DNS server (like Cloudflare 1.1.1.1 or Google 8.8.8.8). Your ISP only sees encrypted traffic — they cannot see which domains you are looking up.

What is a DNS Leak?

A DNS leak happens when your device accidentally sends DNS queries outside the VPN tunnel, directly to your ISP. This defeats the purpose of the VPN because the ISP can still see which websites you visit. Good VPN apps prevent this!
