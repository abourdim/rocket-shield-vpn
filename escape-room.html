What you learned

  • Caesar cipher — a simple shift substitution. Trivially broken today but foundational.
  • Common ports — well-known TCP/UDP numbers. HTTPS is 443.
  • Base64 — binary to text encoding using 64 characters. Not encryption!
  • WireGuard config — defines Interface (local) and Peer (remote) sections.
  • Private keys — WireGuard keys are 32 bytes encoded in base64 (44 chars with padding).
