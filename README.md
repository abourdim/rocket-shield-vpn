## 🌟 What Is This?

**Rocket Shield VPN** is a single-file web app designed for **kids aged 6–15** (and curious adults!) to learn about VPNs, encryption, internet safety, and cybersecurity through a fun space-themed experience.

It's **not a real VPN** — it's an educational tool that simulates how a VPN works and teaches the concepts behind it using:

- 🚀 **Space metaphors** kids actually understand
- 🧪 **Quizzes** to test knowledge
- 🤖 **ByteBot**, an animated robot mascot that reacts to everything you do
- 🎮 **Gamification** with badges and achievements
- 🌍 **Three languages** — English, French, and Arabic (with RTL support)

## 🎮 What's Inside?

The app has **three main sections**, accessible through tabs at the top:

### 🚀 1. Command Center

This is your spaceship cockpit! Here you can:

- **Connect to a VPN simulation** — tap the glowing orb or the big "Launch Shields!" button. Watch the launch sequence with countdown animation, then see your stats come alive.
- **See live mission stats** — mission time, data received, data sent, and ping speed. All update in real-time every second while you're "connected."
- **Choose your base** — pick from 10 server locations around the world (New York, London, Tokyo, Paris, and more). Each shows ping time and server load.
- **Customize your ship** — pick from 5 rocket color themes (Flame, Electric, Stealth, Gold, Neon).
- **Mission Control settings** — toggle your Force Field (kill switch), Auto-Launch, Sound FX, and choose your VPN protocol.
- **Threat Radar** — a fun animated radar that shows "Threats detected!" when disconnected and "All clear!" when protected.

## 🚀 Quick Start — Launch Script

The project includes a **Mission Control** launch script with an interactive menu for serving the app and setting up real VPN infrastructure.

### Linux / macOS / Raspberry Pi

```bash
# Make executable and run
chmod +x rocket-shield.sh
bash rocket-shield.sh
```

### Windows (PowerShell)

```powershell
powershell -ExecutionPolicy Bypass -File rocket-shield.ps1
```

### Docker

```bash
# Web app only
docker-compose up -d

# Web app + WireGuard VPN server (edit docker-compose.yml to uncomment wireguard)
docker-compose up -d
```

### Menu Options

| Option | Description |
|--------|-------------|
| 🌐 Launch Web App | Serve `index.html` locally with auto-browser open + QR code for mobile |
| 🔒 VPN Client Setup | Install & configure WireGuard or OpenVPN client |
| 🌍 VPN Server Setup | Host your own WireGuard VPN server (with client QR codes!) |
| 📊 Status & Diagnostics | Check public IP, DNS leaks, VPN status, system info |
| 🛑 Stop All Services | Stop web server and VPN connections |

### 🍓 Raspberry Pi Deployment

The script auto-detects Raspberry Pi and optimizes accordingly:

```bash
# On your Raspberry Pi
git clone https://github.com/abourdim/rocket-shield-vpn.git
cd rocket-shield-vpn
bash rocket-shield.sh
# Select option 3 → WireGuard Server
# It will install WireGuard, generate keys, create client configs with QR codes!
```

**Tips for Pi deployment:**
- Use a static IP or Dynamic DNS
- Forward port 51820/UDP on your Freebox/router
- WireGuard is very lightweight — perfect for Pi Zero/3/4/5
- The web app serves fine on Pi's built-in Python

## 🛠️ Technical Details (For Developers & Teachers)

| Feature | Implementation |
|---------|---------------|
| **Framework** | None — vanilla HTML, CSS, JavaScript |
| **CSS** | Custom properties (variables), animations, keyframes, grid, flexbox |
| **Animations** | Pure CSS — twinkling stars, pulse rings, radar sweep, floating mascot, confetti, card flips |
| **i18n** | Custom lightweight system using data attributes and a JS translation object |
| **RTL** | Full Arabic RTL support with `dir="rtl"` and CSS logical properties |
| **Themes** | CSS custom properties swapped via `data-theme` attribute |
| **Responsive** | Mobile-first with media queries at 640px breakpoint |
| **Logo** | Base64-embedded JPEG (no external files needed) |
| **Accessibility** | Semantic HTML, focusable buttons, high contrast text |
| **File size** | ~120KB total (including embedded logo) |
| **Dependencies** | Zero. Only Google Fonts (Fredoka + Nunito + Amiri) loaded via CDN |

## 🧑‍🏫 For Teachers & Parents

This app was designed to be used in:

- **Classroom lessons** about internet safety and cybersecurity
- **After-school coding clubs** as a demo of what HTML/CSS/JS can do
- **Home learning** for curious kids who want to understand how the internet works
- **Library workshops** about digital literacy
- **Cybersecurity awareness events** for young people

**Tips for using it in class:**

1. Start with the **Newbie** mode in Mission Academy — go through each card as a group discussion
2. Let kids explore the **Command Center** to understand the VPN simulation
3. Challenge older students to complete the **Advanced Commander** cards and earn the badge
4. Use the **Launch Pad** section as homework — have kids test a real VPN at home with a parent
5. Ask: "Can you find all the easter eggs?" — it's a great engagement hook!

**No internet required after loading** — once the page loads, everything works offline (except Google Fonts, which will fallback to system fonts).
