How Encryption Protects You

Symmetric Encryption

  • Same key encrypts and decrypts
  • Super fast for bulk data
  • Used inside VPN tunnels
  • Examples: AES-256, ChaCha20

Asymmetric Encryption

  • Public key + private key pair
  • Slower, used for key exchange
  • Lets strangers communicate safely
  • Examples: RSA, Curve25519

What VPNs Use

  • WireGuard: ChaCha20-Poly1305
  • OpenVPN: AES-256-GCM
  • Key exchange: Curve25519
  • Both are military-grade!

Why It Matters

  • Without it, WiFi is an open book
  • Anyone on the network can read you
  • Encryption = sealed envelope
  • No encryption = postcard
